Cloudprint packaging for Debian.

This repository contains packing files for incorporating cloudprint into Debian and derivatives. A number of bug fixes and enhancements are included.

It provides a proxy service that makes locally-defined CUPS printers available for use by approved Google users, both locally and remotely.
